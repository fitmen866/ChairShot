# 椅背投石机

一个纯 Web 端 2D 搞笑小游戏：按住蓄力让椅背后仰，松手后角色先坐起来，椅背再快速回弹，把泡面发射出去击中电脑屏幕。

## 运行

```bash
npm run dev
```

然后打开终端中显示的本地地址。项目也支持直接打开 `index.html` 试玩。

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/`，该目录不提交到 git。

## 内容

- 原生 Canvas + JavaScript，无外部运行时依赖。
- 支持鼠标、触摸和空格键蓄力/发射。
- 使用 AI 生成并本地抠图的卡通素材，最终透明素材保存在 `assets/`。
- 游戏状态包含蓄力、坐起过渡、椅背回弹、泡面飞行、命中和失败反馈。
